One-time setup — takes about 3 minutes.
Step 1 — Create the database

Go to supabase.com → New Project → open SQL Editor and run this:

create table if not exists wg_config ( id text primary key, data jsonb not null, updated_at timestamptz default now() ); create table if not exists chores ( id bigint primary key, data jsonb not null, updated_at timestamptz default now() ); create table if not exists expenses ( id bigint primary key, data jsonb not null, updated_at timestamptz default now() ); create table if not exists rides ( id bigint primary key, data jsonb not null, updated_at timestamptz default now() ); create table if not exists shopping ( id bigint primary key, data jsonb not null, updated_at timestamptz default now() ); alter table wg_config enable row level security; alter table chores enable row level security; alter table expenses enable row level security; alter table rides enable row level security; alter table shopping enable row level security; create policy "Public" on wg_config for all using (true) with check (true); create policy "Public" on chores for all using (true) with check (true); create policy "Public" on expenses for all using (true) with check (true); create policy "Public" on rides for all using (true) with check (true); create policy "Public" on shopping for all using (true) with check (true);
Step 2 — Enter your credentials

Supabase project → Settings → API → copy Project URL and anon key.
